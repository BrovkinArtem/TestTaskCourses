# TestTaskCourses — Карточный каталог курсов

Адаптивная карточная сетка курсов с живым поиском по названию и фильтрацией по категориям. Сделано в рамках тестового задания.

## Технологии
- HTML (BEM-структура)
- SCSS → CSS (в проекте присутствуют исходники .scss и собранный main.css)
- Vanilla JavaScript (мобильный/десктопный UI, фильтрация, live search)
- Статический фронтенд — без сборщиков/фреймворков

## Реализация
- БЭМ-именование классов, модульные стили в blocks/
- Адаптив «резиновый» — макет плавно масштабируется для ширин от 320px до 1920px
- Live search: фильтрация карточек по тексту заголовка без перезагрузки страницы
- Фильтры по категориям реализованы через вкладки (tablist) — карточки сгруппированы по категориям
- Вёрстка учитывает базовую доступность (ARIA-атрибуты для вкладок, семантические теги)

## Как запустить локально
Проект — статический. В корне репозитория:
- Открыть index.html напрямую в браузере (поддерживаются относительные пути к ресурсам)
- Или запустить локальный сервер:
  - С помощью Python:
    - Откройте PowerShell в папке проекта и выполните:
      python -m http.server 8080
    - Перейдите в браузере на http://localhost:8080
  - Или с помощью npm-пакета serve:
    - npm i -g serve
    - serve .
